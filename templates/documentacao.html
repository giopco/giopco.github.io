Documentação da API de Previsão do Tempo

Obter Previsão do Tempo

Para obter a previsão do tempo de uma cidade, faça uma solicitação GET para a seguinte URL: 

/previsao?cidade=NOMEDACIDADE

Substitua "NOMEDACIDADE" pelo nome da cidade para a qual você deseja obter a previsão do tempo.

Parâmetros da Solicitação

  • cidade (obrigatório): O nome da cidade para a qual deseja obter a previsão do tempo.

Exemplo de Solicitação

Exemplo de solicitação GET para obter a previsão do tempo de São Paulo:

/previsao?cidade=SaoPaulo

Resposta da API

A API retornará uma resposta no formato JSON contendo informações sobre a previsão do tempo da cidade selecionada. Aqui está um exemplo de resposta:

    {
        "city": {
            "name": "Sao Paulo",
            "country": "BR"
        },
        "list": [
            {
                "dt": 1633860000,
                "main": {
                    "temp": 25.2,
                    "humidity": 63
                },
                "weather": [
                    {
                        "description": "Clear"
                    }
                ]
            },
            // Outras informações de previsão
        ]
    }

Autenticação

A autenticação é necessária para obter a previsão do tempo. Você deve incluir sua API Key na solicitação como um parâmetro "api_key".

Exemplo de Solicitação Autenticada

Exemplo de solicitação GET com autenticação para obter a previsão do tempo de São Paulo:

/previsao?cidade=SaoPaulo&api_key=SUA_API_KEY

Códigos de Resposta

  • 200 OK: A solicitação foi bem-sucedida, e a previsão do tempo foi retornada.
  • 400 Bad Request: Parâmetros ausentes ou inválidos na solicitação.
  • 401 Unauthorized: Falha na autenticação, API Key ausente ou inválida.
  • 500 Internal Server Error: Erro interno do servidor ao obter a previsão do tempo.
