get /cidade
Página de Previsão do Tempo para uma Cidade Específica (cidadeGet)
Renderiza a página para selecionar o estado e a cidade para verificar a previsão do tempo.

Query parameters

estado (required)
Query Parameter — Estado selecionado.
cidade (required)
Query Parameter — Cidade selecionada.

Responses

200

Página de seleção de cidade carregada com sucesso.
